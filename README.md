## EV Range Analyzer

This project was completed as part of a group academic project focused on analyzing factors that affect electric vehicle range and usability.

The program uses Python and API-based data sources to collect and analyze:

- Location data
- EV charging infrastructure
- Elevation
- Weather conditions
- Traffic flow
- EV suitability scoring

## Skills Demonstrated

- Python programming
- API requests and responses
- Data handling
- Error handling
- Debugging
- Pytest testing
- Mocking API responses
- Technical problem-solving
- Team collaboration

## My Contribution

This was a group academic project. My main contributions focused on the API and analysis side of the project, specifically working with elevation data, weather condition data, and supporting the EV sustainability scoring logic.

I helped connect real-world inputs, such as elevation and weather conditions, to the overall EV suitability score so the project could show how different environmental factors may affect electric vehicle range and usability.
